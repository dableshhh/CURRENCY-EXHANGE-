# CURRENCY-EXHANGE-
THIS PROJECT SHOWS CURRENT VALUE OF THE MONEY IN DIFFENT COUNTRY
This project demonstrates integration with exchange rate APIs and provides a clean, user‑friendly interface for quick conversions.
<img width="572" height="405" alt="Screenshot 2026-01-30 180715" src="https://github.com/user-attachments/assets/d1de4ba8-a380-41bd-b9b1-172d353df897" />

Features
Real‑time currency conversion using live exchange rates

Support for multiple currencies (USD, EUR, INR, GBP, etc.)

Intuitive user interface for quick conversions

Historical exchange rate lookup (optional)

Responsive design for desktop and mobile

Easy to integrate into other applications
